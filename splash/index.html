This plugin was created to import games owned in the Amazon Games App.

Allegedly this app could replace the official Twitch App to access games redeemed through Twitch Prime, although the latter is still supported.

As this plugin does not access any API endpoints, it has the following requirements:

  • The Amazon Games App is installed
  • You have successfully authenticated yourself in the app (to populate the games list)

Warning: Although the plugin searches for the installed application, it does not check whether you have been successfully authenticated.
